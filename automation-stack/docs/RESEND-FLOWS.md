# RUBAN Resend Flows

## First email flows to build

### 1. Access Request Confirmation
Trigger:
- user requests access details

Contains:
- confirmation
- what happens next
- private delivery expectations

### 2. Premium Onboarding Instructions
Trigger:
- user purchases or is approved manually

Contains:
- access instructions
- member room / Discord instructions
- support fallback

### 3. Event Reminder / Update
Trigger:
- fight week or event day

Contains:
- event reminder
- where to access board
- timing expectations

## Rule
Keep tone premium, concise, and operational.
